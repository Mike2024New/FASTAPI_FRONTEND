Объявление переменных и использование их в шаблоне

{% set variable = "test val" %} {% set num = 123 %} {% set arr = [1,2,3,4,5] %}

Определенная переменная variable: {{ variable }}

Определенная переменная num: {{ num }}

Определенная переменная arr: {{ arr }}

Объект namespace -> контейнер глобальных переменных

{% set ns_vars = namespace(name="Иван",age=32,city="Москва",) %}

Имя: {{ ns_vars.name }}

Возраст: {{ ns_vars.age }}

Город: {{ ns_vars.city }}

Изменение переменной в цикле

{% set ns = namespace(count=0) %} {% for i in range(10) %} {% set ns.count = ns.count + 1 %} {% endfor %}
