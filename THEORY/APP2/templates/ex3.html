Определение типов (тесты типов)

Проверка, что переменная была определена

{{ "переменная abc определена" if abc is defined else "переменная abc не определена" }}

С помощью "is defined" можно проверить была ли определена переменная или нет

Разбор элементов в тернарных операторах

  • Булевый тип: {{ "true" if arr.bool is boolean else "false" }}
  • Строка: {{ arr.string if arr.string is string else "не строка" }}
  • Число: {{ arr.num if arr.num is integer else "не число" }}
  • Список: {{ arr.list if arr.list is sequence else "не список" }}
  • Словарь: {{ arr.mapping if arr.mapping is mapping else "не словарь" }}
  • Функция: {{ arr.callable if arr.callable is callable else "не функция" }}

Можно прямо в шаблоне с помощью is *type* определять тип переменной
